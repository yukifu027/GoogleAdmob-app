# 概要
GoogleAdMobを使って、広告（現在はテスト用）を表示させることができます。広告の種類は、バナー広告とインタースティシャル広告の2種類です。

# 制作期間
1.5日間

# 制作背景(意図)
これからビジネス視点でも開発できるエンジニアを目指すにあたり、広告表示というビジネスロジックを実装することは必要だと思いました。
